### 1. Portfolio Website

* **Folder Link:** `[View Code](./1-portfolio-website/)`

### 2. SaveTrack App
A robust personal finance utility designed to track income, expense and savings goals. This project focuses heavily on strict accessibility standards and clean dashboard layouts.
* **Core Features:** Savings goal calculator, liquid asset summary dashboard, and user-friendly financial forms.
* **Design Focus:** WCAG accessibility standards, typography hierarchy, and a clean, high-contrast UI.
* **Folder Link:** `[View Project](./2-savetrack-app/)`

### 3. PetrolTrack App
A specialized mobile application concept built to monitor vehicle fuel efficiency, track changing fuel price trends, and calculate monthly vehicle spending.
* **Core Features:** Clean summary circles for fast data reading, interactive price trend graphs, and multiple vehicle management.
* **Design Focus:** Grayscale structural wireframing, specialized typography pairing (**Jost** and **Quicksand**), and dark-mode optimization.
* **Folder Link:** `[View Project](./3-petroltrack-app/)`
